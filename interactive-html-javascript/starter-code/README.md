# Interactive HTML + JavaScript - Starter Code

Welcome to your interactive message box mission!

## Getting Started

1. Fork this repository on GitHub
2. Clone to your computer
3. Open in VS Code
4. Open the integrated terminal (View -> Terminal or Ctrl+`)

## Files in This Workshop

- `index.html` - The HTML structure and inputs
- `styles.css` - The styles and theme classes
- `script.js` - The JavaScript you will complete

## Run the Workshop

Open `index.html` in your browser.

Tip: Keep the file open and refresh after each save.

## Your Mission

Open `script.js` and follow the TODO comments to:

- Select DOM elements
- Update the preview text and styles
- Add event listeners for live updates
